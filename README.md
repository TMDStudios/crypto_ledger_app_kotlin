# crypto_ledger_app_kotlin
Crypto Ledger App (Kotlin) is an extension of the Crypto Ledger website. It makes use of the Crypto Ledger API to buy, sell, and monitor coins. The app was built using Kotlin and Android Studio. It uses coroutines and retrofit to make API calls, and fragments for navigation.

You can see the website here: http://crypto-ledger.herokuapp.com/

The website repository is available here: https://github.com/TMDStudios/crypto_ledger
